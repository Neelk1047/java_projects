# Student Management System

A console-based Java project using JDBC and MySQL for managing student records.

## Features

- Add New Student
- View All Students
- Search Student by ID
- Update Student Details
- Delete Student Record
- Exit Program

## Technologies Used

- Java
- JDBC
- MySQL
- Eclipse IDE

## Project Structure

src/com/student/

- Main.java
- DBConnection.java
- StudentOperations.java

## Database Used

- Database Name: studentdb
- Table Name: students

## Concepts Used

- OOP (Classes & Objects)
- JDBC Connectivity
- SQL CRUD Operations
- PreparedStatement
- ResultSet
- Exception Handling

## How to Run

1. Create database and table in MySQL.
2. Add MySQL JDBC Driver in Eclipse.
3. Run Main.java file.
4. Use console menu to manage student records.
